# auther 文武双全

本脚本主要用于基于阿里云Debian7.5的反向代理服务器上安装Tengine。
#脚本使用方法
将install.sh文件上传到服务器的任意目录，然后执行sh install.sh即可。tengine采用默认安装和默认配置，安装路径在/usr/local/。
更多Tengine的使用方法，请参考Tengine的官网http://tengine.taobao.org/。
